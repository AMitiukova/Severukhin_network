# **Сетевой анализ художественных объединений России и СССР (1820–1932)**

## Учебный DH-проект магистратуры Европейского университета

## **Описание проекта**

Проект по созданию сетевой модели деятелей культуры и художественных объединений Российской империи и Советского Союза реализован в рамках курса «Введение в Digital Humanities» для магистрантов программы «Современные музейные стратегии и кураторские практики» Европейского университета в Санкт-Петербурге.

Курс был разделён на семь тематических блоков, включающих работу с неструктурированными данными, OCR и обработку текста, текстовый анализ и разметку, сетевой анализ, геоинформационные системы (GIS), цифровые коллекции и архивы, Связанные Открытые Данные (LOD), а также инструменты автоматизации работы с библиографией.

Материалом для проекта стал справочник
**Д. Я. Северюхина и О. Л. Лейкинда
«Золотой век художественных объединений в России и СССР (1820–1932)»**,
включающий сведения более чем о 300 художественных объединениях: их участниках, датах существования, географии и структуре.

Студенты проанализировали текст справочника, разработали варианты проектных решений, структурировали данные и создали цифровой ресурс, позволяющий исследовать художественные объединения и их участников автоматизированными методами DH-сообщества.

## **Данные**

Исходные материалы справочника были преобразованы в структурированную таблицу. В репозиторий загружены:

* PDF-A версия справочника (`dataset/`)
* основной датасет
* вспомогательные датасеты для структурирования, нормализации и сетевого анализа

### **Основной датасет: `main_table_dataset_v1.csv`**

| Поле                            | Описание                                         | Формат      |
| ------------------------------- | ------------------------------------------------ | ----------- |
| **Название группы/объединения** | Полное наименование организации на русском языке | text        |
| **Аббревиатура названия**       | Краткое обозначение, если имеется                | text        |
| **Начало работы**               | Год возникновения                                | date (YYYY) |
| **Конец работы**                | Год прекращения деятельности                     | date (YYYY) |
| **Описание группы/объединения** | Полный текст описания из справочника             | text        |
| **Основные участники группы**   | Перечень участников, приведённый к единому виду  | text        |
| **Официальное / неофициальное** | Статус объединения                               | text        |
| **Территориальность**           | Историческое название локации                    | text        |
| **Современные названия**        | Актуальное название города/страны                | text        |
| **Latitude / Longitude**        | Географические координаты                        | float       |
| **Библиография**                | Ссылки на источники и литературу                 | text        |

### **Вспомогательные датасеты**

* **`surnames.csv`** — структурированный именной указатель справочника (ФИО + страницы).
* **`Bibliography.csv`** — полные библиографические записи, соответствующие сокращённым ссылкам.
* **`nodes_edges_table.csv`** — таблица рёбер для сетевого анализа (связи «участник → объединение»).

## **Сетевой анализ**

Сетевой анализ выполнен в редакторе **Gephi**.

* Узлы: фамилии участников и названия объединений.
* Раскладка выполнена алгоритмом **ForceAtlas2**.
* Визуальные параметры узлов (размеры подписей) отражают показатели центральности.
* Проведена автоматическая кластеризация на сообщества с цветовой маркировкой.
* Для каждого узла рассчитаны основные показатели центральности.

Интерактивный граф загружен в репозиторий в виде HTML-страницы.
Реализован поиск по фамилиям и названиям объединений.
При выборе узла отображаются сведения о его центральности и связях:

* для участника — все объединения, в которых он состоял;
* для объединения — полный список участников.

## **Дальнейшая работа**

1. **Ручная валидация основного датасета**
   Проверка корректности распознавания текста, приведение имён участников к единому виду, сверка с именным указателем.

2. **Связывание данных с внешними словарями и тезаурусами**

   * сопоставление участников с сущностями **Wikidata**,
   * сопоставление локаций с записями **Geonames**.

3. **Доработка библиографии**
   Расшифровка сокращённых ссылок и приведение их к полному формату.

4. **Географическая визуализация**
   Создание карты распределения художественных объединений по территории России и СССР, включая статистику по числу участников.

5. **Динамическая визуализация (temporal network)**
   Отображение временных интервалов существования объединений и анализ периодов активизации художественных инициатив.

6. **Библиотека Zotero**
   Собрана коллекция источников и литературы из справочника для удобной работы исследователей.

## **Ценность проекта**

Оцифровка и структурирование сведений справочника делают уникальный массив данных доступным для анализа и использования.
Опубликованный граф и датасеты предоставляют исследователям, музейным специалистам и DH-сообществу:

* инструменты сетевого анализа художественных объединений;
* возможность выявления центральных фигур и ключевых взаимодействий;
* основу для расширения исследования и включения дополнительных данных;
* удобный цифровой доступ к информации, ранее представленной только в печатном формате.
