### Краткое описание
Проект по созданию сетевой модели деятелей культуры и художественных объединений Российской империи и СССР реализован в рамках курса **«Введение в Digital Humanities»** для магистров программы **«Музейные исследования и кураторские стратегии»** Европейского университета (Санкт-Петербург).

Курс включал семь тематических блоков: работа с неструктурированными данными, OCR и обработка текста, текстовый анализ и разметка, сетевой анализ, геоинформационные системы (GIS), цифровые коллекции и архивы, Связанные Открытые Данные (LOD) и инструменты автоматизации библиографии.

Материалом проекта стал справочник:

**Д. Я. Северюхин, О. Л. Лейкинд** — *Золотой век художественных объединений в России и СССР (1820–1932)*, содержащий сведения о более чем 300 объединениях, их участниках, хронологии и географии.

## Сетевой анализ и визуализация

Аналитическая часть выполнена в **Gephi**:

- Узлы: фамилии участников + наименования объединений.  
- Рёбра: участие (связь «участник → объединение»).  
- Раскладка: алгоритм **ForceAtlas2**.  
- Кластеризация: автоматическое выделение сообществ; каждому сообществу назначен цвет.  
- Визуальные метки и размеры узлов соответствуют показателям центральности (degree, betweenness и т.п.).  
- Рассчитаны основные метрики центральности для каждого узла.

Интерактивный граф и сопутствующие визуализации публикуются через **GitHub Pages** — ссылка на деплой проекта указана в шапке README (замени на реальный URL). Файлы, необходимые для сборки страницы, находятся в `viz/` и/или в корне ветки, настроенной для Pages.

При просмотре через Pages реализованы:

- поиск по фамилиям и по названиям объединений;  
- отображение связей и показателей центральности при выборе узла;  
- подсчёт числа участников в объединении и числа объединений у участника.

## Текущий статус и дальнейшие шаги

На настоящий момент опубликована первая версия датасета и графа. План дальнейших работ:

1. **Ручная валидация данных** — исправление ошибок OCR, унификация имён, сверка с именным указателем.  
2. **Связывание с контролируемыми словарями** — сопоставление участников с **Wikidata**, локаций — с **Geonames**.  
3. **Доработка библиографии** — расшифровка сокращённых ссылок и нормализация записей.  
4. **Географическая карта** — визуализация распределения объединений по территории (карта + статистика по числу участников).  
5. **Динамическая визуализация** — создание temporal network для отображения временных интервалов существования объединений.  
6. **Публикация данных** — подготовка окончательного датасета и депозита на Zenodo (DOI).  
7. **Zotero** — коллекция источников подготовлена для дальнейшего использования.
